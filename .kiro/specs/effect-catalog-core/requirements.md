# Requirements: Effect Catalog Core

Parent spec: #[[file:.kiro/specs/animation-effects-library/requirements.md]]

This spec covers the foundational catalog infrastructure (Requirements 1-8, 17-19 from the parent). It includes the EffectSkeleton data model, EffectCatalog persistence, EffectRegistry dispatch, CodegenEngine refactor, LegacyMapper, sync_points, quality_profiles, Effects API, Effect Browser UI, save-as-effect workflow, and serialization round-trip.

## Included Requirements

- Requirement 1: Effect Skeleton Definition
- Requirement 2: Effect Catalog Persistence
- Requirement 3: Effect Registry and Dispatch
- Requirement 4: Codegen Engine Integration
- Requirement 5: Effect Browser API
- Requirement 6: Effect Browser UI
- Requirement 7: Effect Extraction from Existing Videos
- Requirement 8: Effect Skeleton Serialization Round-Trip
- Requirement 17: Narration-Sync Anchors
- Requirement 18: Hardware-Aware Render Profiles
- Requirement 19: Legacy Type Mapping

Full acceptance criteria for each requirement are defined in the parent spec.
